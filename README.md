## 🎯 О проекте

Представь, что у тебя есть личный репетитор по физике, который всегда под рукой, прямо в Telegram. Он не злится, если ты что-то не понимаешь, и не будет решать за тебя. Вместо этого он поможет разобраться в сложных темах и научит решать задачи самостоятельно. Как друг, который хочет, чтобы ты сам всё понял!

Этот бот создан специально для школьников, которые готовятся к экзаменам по физике. Он работает в Telegram, поэтому ты можешь использовать его на телефоне, планшете или компьютере. Он совсем как обычный собеседник.

## ✨ Особенности

- 🤝 **Не решает за тебя** – помогает понять и научиться
- 📱 **Всегда под рукой** – работает в Telegram на любом устройстве
- 🎯 **Удобная навигация** – всё управляется кнопками
- 💡 **Понятные объяснения** – сложные вещи простыми словами
- 📊 **Отслеживает прогресс** – помнит, что ты уже изучил

## 🚀 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |

## 📋 Основные разделы

### 1️⃣ Помощь с задачей
Застрял на задаче? Выбери тему (механика, электричество и др.) и отправь условие. Бот не даст готовый ответ, а подскажет:
- Какую формулу использовать
- Что нужно найти в первую очередь
- Как проверить правильность решения

### 2️⃣ Лабораторные работы
Вся информация для подготовки к экспериментальным заданиям:
- Список оборудования
- Примеры оформления
- Советы экзаменаторов
- Типичные ошибки

### 3️⃣ Теория
Забыл закон Ома? Краткие и понятные объяснения:
- Простыми словами
- С формулами и обозначениями
- Примеры применения

### 4️⃣ Учёные
Интересные факты о жизни и открытиях:
- Ньютон
- Эйнштейн
- Ломоносов
- И другие

### 5️⃣ Частые вопросы
Ответы на волнующие вопросы:
- Как готовиться к ЕГЭ?
- Сколько баллов нужно?
- Нужен ли репетитор?

### 6️⃣ Избранное
Сохраняй важные задачи и теорию, чтобы быстро возвращаться к ним позже.

### 7️⃣ Статистика
Отслеживай свой прогресс:
- Сколько раз открывал меню
- Какие разделы изучал
- И другие метрики

## 🛠 Технологии

- Python 3.7+

- python-dotenv

## 📦 Установка и запуск

1. **Клонируй репозиторий:**
```bash
git clone https://github.com/yourusername/physics-tutor-bot.git
cd physics-tutor-bot
```

2. **Установи зависимости:**
```bash
pip install -r requirements.txt
```

3. **Создай файл `.env` и добавь токен:**
```env
BOT_TOKEN=твой_токен_бота
```

4. **Запусти бота:**
```bash
python main.py
```
